# 前端项目全局智能体规则（TRAE全局生效）
## 1. 开发流程约束
- 遵循最小化修改原则，不随意重构存量代码
- 代码生成后自动合规校验，不符合规范自动修正
- 全程保留需求溯源、变更记录

## 2. 工程规范
- 技术栈：Vue3 + TS + Tailwind CSS + Element Plus
- 目录结构：src/views/ 页面、src/components/ 公共组件、src/hooks/ 通用钩子、src/utils/ 工具函数
- 编码规范：ES6+、严格TS类型、组件props/emits完整定义、scoped样式
- 兼容性：移动端响应式、W3C标准、无原生DOM暴力操作

## 3. AI行为约束
- 优先遵循本文件全部规则
- 最小化文件修改，不随意重构存量代码
- 代码生成后自动合规校验，不符合规范自动修正
- 全程保留需求溯源、变更记录
