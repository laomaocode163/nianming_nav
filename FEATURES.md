# Nianming Nav - 项目功能说明文档

## 项目概述

**Nianming Nav** 是一个简洁高效的个人导航站，帮助用户管理和快速访问常用网站。基于现代前端技术栈构建，支持本地数据管理和云端同步。

---

## 核心功能模块

### 1. 首页导航功能

#### 1.1 多引擎搜索
- **快捷键支持**: `⌘K` (Mac) / `Ctrl+K` (Windows/Linux) 快速聚焦搜索框
- **多搜索引擎**: 支持 Google、Bing、百度、GitHub 等主流搜索引擎
- **一键切换**: 点击搜索引擎标签即可切换默认搜索源
- **实时搜索**: 输入关键词后按回车即可在新标签页打开搜索结果

#### 1.2 分类浏览
- **侧边栏导航**: 左侧分类栏展示所有网站分类
- **全部网站**: 支持查看所有分类下的网站
- **分类筛选**: 点击分类只显示该分类下的网站
- **数量统计**: 每个分类显示包含的网站数量

#### 1.3 网站卡片展示
- **图标显示**: 自动获取网站 favicon 图标
- **网站信息**: 显示网站名称、描述、所属分类
- **置顶标识**: 置顶网站有特殊视觉标识
- **点击跳转**: 点击卡片直接打开网站（新标签页）

#### 1.4 侧边栏交互
- **展开/收起**: 支持完全收起侧边栏
- **折叠模式**: 支持折叠为图标模式（仅显示图标）
- **移动端适配**: 移动端自动变为抽屉式侧边栏
- **快捷入口**: 底部管理后台入口

---

### 2. 批量编辑功能

#### 2.1 批量编辑模式
- **快捷键**: `Esc` 退出批量编辑模式
- **全选**: `⌘A` / `Ctrl+A` 全选当前分类网站
- **取消全选**: 一键取消所有选择

#### 2.2 批量操作
- **批量置顶/取消置顶**: 对选中的网站进行置顶状态切换
- **批量删除**: 一键删除选中的多个网站（带确认提示）
- **批量移动**: 将选中的网站移动到其他分类
- **选择计数**: 实时显示已选择的网站数量

#### 2.3 排序模式
- **拖拽排序**: 支持拖拽调整网站显示顺序
- **快捷键**: `Esc` 退出排序模式
- **视觉反馈**: 拖拽时有阴影和边框提示
- **自动保存**: 排序完成后自动保存

---

### 3. 管理后台功能

#### 3.1 网站管理

**添加网站**
- 网站名称（必填）
- 网站地址（必填，自动补全 https://）
- 所属分类选择
- 网站描述（可选）
- 置顶设置
- 自定义图标（可选）

**图标管理**
- **自动获取**: 输入 URL 后自动获取网站 favicon
- **手动上传**: 支持上传 PNG、JPG、SVG、ICO 格式图标（最大 2MB）
- **图标预览**: 实时预览图标效果
- **自动获取开关**: 可关闭自动获取功能

**编辑网站**
- 修改所有网站信息
- 更换图标
- 更改所属分类
- 置顶状态切换

**删除网站**
- 单个删除（带确认提示）
- 显示网站所属分类
- 置顶网站特殊标识

**重复检测**
- 同一分类内重复链接检测
- 跨分类重复链接提示
- 确认后允许跨分类添加

#### 3.2 分类管理

**添加分类**
- 分类名称
- 分类图标（15 个预设图标可选）

**编辑分类**
- 修改分类名称
- 更换分类图标

**删除分类**
- 带确认提示
- 自动将网站移动到默认分类
- 至少保留一个分类

**分类统计**
- 显示每个分类下的网站数量

#### 3.3 系统设置

**站点设置**
- 网站标题自定义
- 导航标题自定义
- 卡片样式切换（详细模式/简洁模式）
- 站点模式切换（个人模式/站长模式）

**数据管理**
- **导出数据**: 将数据导出为 JSON 文件备份
- **导入数据**: 从 JSON 文件恢复数据
- **重置数据**: 一键恢复默认数据（带确认提示）

**数据统计**
- 分类数量统计
- 网站数量统计
- 置顶网站数量统计

---

### 4. 数据管理功能

#### 4.1 本地存储
- **自动保存**: 所有操作自动保存到浏览器 LocalStorage
- **数据持久化**: 刷新页面数据不丢失
- **数据迁移**: 自动从旧版本数据格式迁移

#### 4.2 图标管理
- **域名映射**: 基于域名的图标映射表
- **自动获取**: 使用 faviconextractor API 自动获取图标
- **自定义图标**: 支持 Base64 格式自定义图标
- **清理优化**: 自动清理未使用的图标数据

#### 4.3 云端同步（可选）
- **Cloudflare KV**: 支持 Cloudflare KV 存储同步
- **密码保护**: 同步操作可设置密码保护
- **API 接口**: 提供同步 API 接口

---

### 5. 主题与样式

#### 5.1 主题切换
- **亮色模式**: 清爽明亮的浅色主题
- **暗色模式**: 护眼的深色主题
- **自动检测**: 自动检测系统主题偏好

#### 5.2 响应式设计
- **桌面端**: 侧边栏 + 主内容区布局
- **平板端**: 自适应网格布局
- **移动端**: 单列布局 + 抽屉式侧边栏

#### 5.3 视觉特性
- **卡片设计**: 圆角卡片，阴影效果
- **动画过渡**: 平滑的过渡动画
- **悬停效果**: 丰富的交互反馈
- **加载状态**: 图标获取时的加载动画

## 部署方式

### 方式一：Cloudflare Pages（推荐）
- 自动构建部署
- 全球 CDN 加速
- 免费额度充足

### 方式二：Cloudflare Workers
- 支持自定义域名
- 支持优选 IP
- 国内访问更快

### 方式三：静态托管
- 支持任何静态托管服务
- 纯前端应用，无需服务器

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 下一代前端构建工具
- **Tailwind CSS v4** - 原子化 CSS 框架
- **Pinia** - Vue 状态管理
- **Vue Router** - Vue.js 官方路由
- **Element Plus** - UI 组件库
- **vue-draggable-plus** - 拖拽排序组件
